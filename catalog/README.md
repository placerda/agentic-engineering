# Catalog

activate it. See the
[Agent-Native Engineering guide](../docs/agent-native-engineering.md) for
activation and overlap guidance.

| --- | --- | --- |
| [`issue-triage`](agents/issue-triage.agent.md) | Agent | Clarify, classify, and prioritize issues before implementation |
| [`technical-support`](agents/technical-support.agent.md) | Agent | Diagnose reproducible failures and define a safe recovery path |
| [`documentation-ux`](agents/documentation-ux.agent.md) | Agent | Document confirmed user and operator journeys |
| [Python](instructions/python.instructions.md) | Scoped instruction | Apply project-aware guidance to Python files |
| [Cloud native](instructions/cloud-native.instructions.md) | Scoped instruction | Apply deployment and operation guidance to service and infrastructure files |
