# Catalog

Optional specialist agents. Copy one into `.github/agents/` to activate it.
See the
[Agent-Native Engineering guide](../docs/agent-native-engineering.md) for
activation and overlap guidance.

| Agent | Purpose |
| --- | --- |
| [`issue-triage`](agents/issue-triage.agent.md) | Clarify, classify, and prioritize issues before implementation |
| [`technical-support`](agents/technical-support.agent.md) | Diagnose reproducible failures and define a safe recovery path |
| [`documentation-ux`](agents/documentation-ux.agent.md) | Document confirmed user and operator journeys |
